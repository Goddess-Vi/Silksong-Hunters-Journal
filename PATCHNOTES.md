# Pale Spider's Journal Patch Notes
The official release notes for the Pale Spider's Journal website
___
## v0.1.0 ("First" Version)
#### The site includes the following:
* The portrait of all the creatures in Hollow Knight: Silksong
* A prototype filter system that works however only a select few creatures work within the filter as of right now
* A section for the health of creatures, including black-threaded if possible (Select few have this)
* A section for the locations of enemies (Every creature!)
* A section for the rosary and shell shard drops of creatures
* A section for the creature's in-game description & comment (Only some)
* The creature's name (Only some)
### What's next to come?
The next version of the website will have every creature's name filled out with around 50% of the creature's having their descriptions & comments filled in
